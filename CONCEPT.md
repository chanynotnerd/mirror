# Mirror *(가제)* — 컨셉 브리프

> 한 장짜리 닻 문서. 스코프가 흔들릴 때 "이게 컨셉에 맞나?"로 되돌아오는 기준점.
> 가제 후보: **Mirror / Twin Tap / Echo** — 출시 전 확정.

---

## 한 줄 소개
화면을 한 번 탭하면 위·아래 두 캐릭터가 **동시에** 움직인다. 단 하나의 입력으로 두 곳을 동시에 살려야 하는 반응형 아케이드.

## 핵심 루프 (Core Loop)
탭으로 두 캐릭터 동시 조종 → 어긋나는 장애물 패턴 통과 → 점수·코인 획득 → 충돌로 사망 → **즉시 재시작** → "한 판 더"

## USP (차별점)
- **입력 1개 = 두 곳 동시 관리.** 단순한데 머리가 꼬이는 독창적 컨트롤. 유사 클론이 거의 없음.
- **코인을 위험한 자리에 배치** → "탐욕 vs 생존"의 매 순간 선택. 코인이 단순 수집물이 아니라 도박이 됨.
- **미니멀 비주얼 + 강한 손맛**(햅틱·사운드·파티클)으로 완성도 승부. 메커니즘이 아니라 *실행 퀄리티*가 차별점.

## 타겟
- 짧은 세션(1~3분)을 즐기는 캐주얼 유저
- 하이스코어 도전형 + 스킨 수집형 둘 다 포섭

## 플랫폼 / 기술
- iOS / Android, **모바일 세로 모드**
- **Expo (React Native)** + EAS Build로 스토어 제출
- 1인 개발 + Claude Code

## 목표
- **1차 목표: 스토어 출시** (완성하고 제출하는 경험)
- 차트 진입이 아니라 "끝까지 출시한 완성품"이 성공 기준
- 관전 지표: 핵심 루프 완성도, D1 리텐션

## 스코프 (MVP)
- 모드: 패럴렐(기본) + 인버티드(하드, 30점 달성 시 해금)
- 화면 3개: 타이틀 / 게임 / 게임오버
- 메타: 코인 + 스킨 2~3종
- 수익: 전면·보상형 광고 + 광고 제거 IAP
- **제외(v1.1로):** 데일리 챌린지, 리더보드, 추가 테마
